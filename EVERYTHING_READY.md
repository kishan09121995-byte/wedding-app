## 🆘 Troubleshooting

| Issue | Fix |
|---|---|
| "npm: command not found" | Install Node.js from nodejs.org |
| "Port 3000 in use" | `npm run dev -- --port 3001` |
| Blank page | F5 to refresh |
| No guests showing | Restart dev server (Ctrl+C → `npm run dev`) |
| Login fails | Check Supabase console for errors |
